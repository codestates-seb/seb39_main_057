## 2.개요
    - 프로젝트 명칭: 호박공구마
    - 개발 인원: Back-End[3명], Front-End[2명]
    - 주요기능: 배달을 위한 가게정보와 개인위치정보 공유, 배달 공구를 진행하여 개인부담 배달비의 절감
    - 개발 언어: Java, javaScript, html, css, aws
    - 개발 환경: Spring FreameWork, React
    - 데이터베이스: mysql
    - 형상관리 툴: github
